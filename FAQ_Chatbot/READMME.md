# FAQ Chatbot using NLP

## Description
A GUI-based FAQ Chatbot developed using Python, Tkinter, and NLP techniques.

## Features
- Answers frequently asked questions
- Uses TF-IDF Vectorization
- Uses Cosine Similarity for matching questions
- GUI built using Tkinter
- Interactive chatbot interface

## Technologies Used
- Python
- Tkinter
- Scikit-learn

## NLP Concepts
- TF-IDF Vectorizer
- Cosine Similarity

## How to Run

pip install scikit-learn

python chatbot_gui.py
