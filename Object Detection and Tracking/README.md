# Object Detection and Tracking using YOLOv8

## Description
This project uses YOLOv8 and OpenCV to detect objects from a video file.

## Features
- Real-time object detection
- Detects persons, cars, buses, motorcycles, backpacks, and more
- Processes video frame by frame
- Saves output video with bounding boxes

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics

## How to Run

pip install ultralytics
pip install opencv-python

python object_detection.py
